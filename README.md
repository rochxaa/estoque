# Controle de Estoque — Python + Tkinter + SQLite

Um sistema simples e funcional de **controle de estoque**, desenvolvido em **Python**, com **interface gráfica (Tkinter)** e **banco de dados SQLite**.  
Permite o **cadastro, listagem e gerenciamento de produtos**, além de um **sistema de login** com dois perfis de usuário:

---

## Tecnologias Utilizadas

- **Python**
- **Tkinter** — interface gráfica
- **SQLite3** — banco de dados embutido
- **Hashlib** — para criptografia de senhas
- **OS / pathlib** — manipulação de arquivos e diretórios
